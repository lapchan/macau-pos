Test Barcodes

Scan these EAN-13 codes with the USB scanner against the cashier POS to verify the hardware scanner pipeline. Slot numbers match the order in test-barcodes.sql.

Setup: run test-barcodes.sql on the production DB first. Then open the cashier (logged in, terminal activated, shift open) and scan from this screen. Each scan should show a green banner at the top of the cashier with the product name. Slot 1 corresponds to the first row in the SQL output (ordered by barcode).
